# Age-adjusted machine learning identifies facial skin microbes associated with skin quality in Koreans
This repository provides the source code used to generate all analyses and figures presented in our study on age-adjusted machine learning for identifying facial skin microbes associated with skin quality in a Korean cohort.

The scripts reproduce the complete analytical workflow, including data processing, model construction, statistical analyses, and visualization of final results.

## Computational Environment

All analyses were conducted using the following software versions:

**Python: 3.11.5**

**R: 4.3.1**

Detailed package dependencies and execution instructions are provided within the respective analysis scripts.
